# Crud Using Php
Yeh project maine PHP aur MySQL ka use karke banaya hai jisme complete CRUD operations implement kiye gaye hain. User simple registration form ke through apna data enter karta hai, jo database me save ho jata hai. Design ko clean, modern aur fully responsive banaya gaya hai jisse mobile ya desktop dono par smooth experience milta hai.

Is project me maine front-end ke liye HTML, CSS ka use kiya hai aur back-end ke liye MySQL + PHP ka. Form me validations, duplicate email check, password handling, gender, phone number, address jaise fields add kiye gaye hain. Data insert hone ke baad users ek beautiful table me show hote hain, jahan se unhe edit ya delete kiya ja sakta hai.

Ye project beginners ke liye perfect hai jo real-world PHP applications banana chahte hain. Code simple, clean aur easy to understand rakha gaya hai. Database aur table creation commands bhi project me include hain.

Aage future me isme search bar, pagination, login system aur image upload jaise features bhi add kar sakta hoon.
